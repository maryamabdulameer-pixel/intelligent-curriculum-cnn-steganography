# Dataset split note

The experiments were conducted using a COCO-based cover-secret pair dataset organized into train, validation, and test folders.
The code expects the following folder structure under `coco_20k_stego_pairs/`:

```text
train/covers/
train/secrets/
val/covers/
val/secrets/
test/covers/
test/secrets/
```

The original COCO images are not redistributed in this repository.
